# RD-Agent-like Framework

Learning project for building an agent framework with local LLMs.

## Structure
- src/agent_core: core engine
- tests: tests
- runs: outputs (ignored)
- docs: roadmap & notes

# Learning Roadmap

## Week 1 -- Core Engine

-   LLM abstraction layer
-   Structured output contracts
-   Retry & repair
-   Run tracking
-   Tool execution

## Week 2 -- Autonomous Loops

-   Auto code generation & self-healing
-   Multi-tool agent & environment repair
-   Memory & reflection
-   Evaluation & convergence
-   Mini RD-Agent demo

## Week 3 -- Intelligence Layer

-   Episodic memory
-   Strategy optimization
-   Scoring & benchmarking
-   Failure pattern learning

## Week 4 -- RD-Agent Level System

-   Hypothesis generation
-   Experiment planning
-   Autonomous execution loops
-   Result analysis
-   Self-improvement cycles
